# AI N-Queens Visual Solver

This project now runs on a Python + Flask + HTML/CSS + JavaScript stack.

## Run locally

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Start the Flask app:

```bash
python app.py
```

4. Open `http://127.0.0.1:5000`

## Stack

- Python
- Flask
- HTML
- CSS
- JavaScript

## App routes

- `/` for configuration
- `/solver` for the visualization view
